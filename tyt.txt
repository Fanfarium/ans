Крок 1: Встановлення Ansible
Встановіть Ansible на вашому сервері або локальному комп'ютері. Для цього виконайте наступну команду в терміналі або командному рядку:


sudo apt update
sudo apt install ansible
Переконайтеся, що Ansible встановлено правильно, запустивши команду ansible --version.

Крок 2: Створення інвентаризаційного файлу
Створіть новий файл інвентаризації. Це текстовий файл, в якому ви визначаєте ваші хости (комп'ютери), до яких ви хочете застосовувати Ansible.

[servers]
server1 ansible_host=192.168.1.100
server2 ansible_host=192.168.1.101
Замініть IP-адреси на реальні IP-адреси ваших комп'ютерів.

Крок 3: Створення плейбука для ping-pong
Створіть новий файл плейбука. Плейбук - це файл YAML, який містить список задач, які Ansible повинен виконати на вказаних хостах.

# playbook.yaml

- name: Ping-pong test
  hosts: servers
  tasks:
    - name: Ping servers
      ping:
У цьому плейбуці ми вказуємо Ansible підключатися до групи хостів "servers" і виконувати модуль "ping", щоб перевірити з'єднання з кожним сервером.

Крок 4: Виконання плейбука
Виконайте плейбук за допомогою команди ansible-playbook та вкажіть шлях до вашого плейбука:

ansible-playbook playbook.yaml -i inventory.txt
Після виконання цієї команди Ansible підключиться до кожного хоста, вказаного в інвентаризаційному файлі, і виконає задані завдання.

Результат
Після успішного виконання плейбука ви повинні побачити вивід, який підтверджує успішне встановлення з'єднання з кожним хостом. Це означає, що Ansible успішно виконав ping-pong тест на кожному з ваших серверів.
